# 집중력을 잃은 개발자를 위한 두 가지 도구

늦은 밤, 코드 에디터 앞에 앉아 있지만 집중이 안 됩니다. 슬랙 알림, 유튜브 추천 영상, 끝없는 SNS 피드. 5분만 쉬려다 1시간이 지나갑니다.

혹시 이런 경험 있으신가요?
- "이번엔 정말 집중해야지" 다짐하지만 30분도 못 버팀
- 할 일은 많은데 어디서부터 시작할지 막막함
- 생산성 앱은 많은데 결국 다 안 쓰게 됨

저희도 같은 고민을 했습니다. 그래서 SidequestLab 초창기에 가장 먼저 만든 게 바로 이 두 도구였어요.

## 왜 만들었나요?

SidequestLab은 사이드 프로젝트 실험실입니다. 다양한 아이디어를 빠르게 실험하고, 실제로 쓸 만한 도구를 만드는 게 목표죠.

첫 번째 질문은 "우리 스스로 필요한 게 뭘까?"였습니다. 그 답은 간단했어요.
1. **집중할 수 있는 환경**
2. **할 일을 잊어버리지 않는 방법**

그래서 하루 만에 두 도구를 만들었습니다. 거창한 기능은 없습니다. 딱 필요한 것만 담았어요.

## 뽀모도로 타이머 - 25분 집중, 5분 휴식

### 뽀모도로 기법이란?

25분 동안 집중하고, 5분 쉬는 걸 반복하는 시간 관리 기법입니다. 이탈리아어로 토마토(Pomodoro)를 뜻하는 이름에서 유래했죠. 짧은 집중 시간과 강제 휴식이 핵심입니다.

### 어떤 기능이 있나요?

- **25분/5분 타이머**: 기본 뽀모도로 주기 (25분 집중 → 5분 휴식)
- **시간 커스터마이징**: 나에게 맞는 시간으로 자유롭게 변경 가능
- **세션 카운터**: 오늘 몇 번이나 집중했는지 한눈에 확인
- **알림**: 집중 시간이 끝나면 브라우저 알림으로 알려줌
- **반응형**: 모바일에서도 PC에서도 동일하게 사용 가능

### 이렇게 사용해보세요

1. 페이지 접속 → 바로 25분 타이머 시작
2. 시간 조정 필요하면 우측 설정 버튼
3. 타이머 종료 시 자동으로 휴식 시간 전환
4. 세션 카운터로 하루 생산성 확인

## 투두 앱 - 할 일을 간단히

### 복잡한 건 싫어요

시중에 나온 투두 앱은 기능이 너무 많습니다. 프로젝트 분류, 태그, 우선순위, 반복 일정... 설정하다 지쳐서 결국 안 쓰게 되죠.

저희 투두 앱은 **정말 심플**합니다.

### 핵심 기능

- **할 일 추가**: 텍스트 입력하고 엔터
- **체크 완료**: 클릭 한 번
- **필터링**: 전체/진행중/완료 탭 전환
- **삭제**: 필요 없으면 휴지통 버튼
- **로컬 저장**: 브라우저에 저장되어 새로고침해도 그대로 유지

### 이렇게 사용해보세요

1. 오늘 할 일을 쭉 입력
2. 하나씩 완료하면서 체크
3. 완료된 항목은 "완료" 탭에서 확인
4. 내일은 새로운 마음으로 다시 시작

👉 **GitHub에서 보기**: [https://github.com/namseokyoo/todo-app](https://github.com/namseokyoo/todo-app)

## 두 도구를 함께 쓰면 시너지

아침에 출근하면 이렇게 써보세요.

1. **투두 앱에 오늘 할 일 정리** (2분)
2. **뽀모도로 타이머 시작** (25분)
3. 한 가지 일에만 집중
4. 휴식 시간에 투두 앱 체크
5. 다시 타이머 시작

이 루틴을 반복하면, 하루에 보통 4-6개 뽀모도로 세션을 완료하게 됩니다. 그럼 약 2-3시간의 순수 집중 시간을 확보한 셈이죠.

## 하루 만에 만든 이유

### 빠른 실험, 빠른 검증

SidequestLab의 철학은 "일단 만들고 보자"입니다. 완벽한 기획서 쓰고, 한 달 개발하고, 출시하는 방식이 아니에요.

- 아이디어 떠오름 → 하루 안에 MVP 완성 → 직접 써봄 → 개선 또는 보류

두 도구도 마찬가지였습니다.
- **투두 앱**: 2026년 1월 26일, 1일 완성
- **뽀모도로 타이머**: 2026년 1월 26일, 1일 완성

### 작지만 실용적

거창한 기능은 없습니다. 대신 실제로 매일 씁니다. 그게 가장 중요하다고 생각해요.
