## 이런 고민, 해보셨나요?

"오늘 점심은 짜장면? 짬뽕?"
"여름휴가는 산? 바다?"
"강아지와 고양이 중에 뭐가 더 좋아?"

친구들과 이런 얘기를 나누다가 갑자기 궁금해진 적 있으시죠? 다른 사람들은 어떻게 생각하는지, 내 선택이 다수일까 소수일까?

저희도 같은 고민을 했습니다. 그래서 **Thisor**를 만들었어요.

## Thisor란?

**"이거 vs 저거"** - 스와이프 한 번으로 즐기는 실시간 이지선다 투표 앱입니다.

틱톡처럼 세로로 스와이프하며 재미있는 투표를 구경하고, 마음에 드는 선택지를 탭하면 바로 투표 완료! 결과는 실시간으로 업데이트되어 지금 이 순간 다른 사람들의 선택을 즉시 확인할 수 있습니다.

## 주요 기능

### 🎯 투표 피드 - 스와이프로 즐기는 투표

틱톡 스타일의 세로 스와이프로 다양한 투표를 탐색하세요. 스와이프 한 번에 새로운 투표, 또 한 번에 다음 투표. 끊임없이 재미있는 질문이 펼쳐집니다.

피자 vs 치킨, 여름 vs 겨울, 아침형 인간 vs 저녁형 인간... 생각지도 못한 조합들이 여러분을 기다리고 있어요.

### ⚡ 1초 투표 - 탭 한 번으로 참여 완료

마음에 드는 선택지를 탭하는 순간 투표 완료! 복잡한 절차 없이, 회원가입도 필요 없이, 정말 1초 만에 투표에 참여할 수 있습니다.

투표 후에는 실시간으로 결과 바가 애니메이션으로 나타나며, 다른 사람들의 선택을 즉시 확인할 수 있어요. "어? 나랑 같은 생각을 하는 사람이 이렇게 많아?" 또는 "내가 소수파구나!" 하는 재미를 느껴보세요.

### 🔧 투표 생성 - 친구들에게 물어보고 싶은 게 있다면?

"오늘 저녁 뭐 먹지?" 같은 결정이 어려울 때, 직접 투표를 만들어보세요. 제목과 두 가지 선택지만 입력하면 바로 투표 생성 완료!

투표 시간은 1분부터 1시간까지 설정할 수 있고, 생성된 링크를 친구들에게 공유하면 모두 함께 투표에 참여할 수 있습니다.

### 👤 내 활동 - 내가 만든 투표, 참여한 투표 한눈에

내가 만든 투표는 어떻게 진행되고 있는지, 내가 참여한 투표의 최종 결과는 어떻게 됐는지 궁금하시죠? 내 활동 페이지에서 한눈에 확인하세요.

진행 중인 투표는 실시간으로 결과가 업데이트되고, 종료된 투표는 최종 결과를 보여드립니다.

## 이렇게 사용해보세요

1. **Thisor 접속** - https://thisor-theta.vercel.app
2. **피드 탐색** - 스와이프하며 재미있는 투표 구경하기
3. **투표 참여** - 마음에 드는 선택지 탭하기
4. **결과 확인** - 실시간으로 업데이트되는 투표 결과 보기
5. **투표 만들기** - 친구들에게 물어보고 싶은 투표 생성하고 공유하기

## 기술적 특징

Thisor는 최신 웹 기술을 활용하여 빠르고 부드러운 사용자 경험을 제공합니다:

- **Next.js 16 & React 19**: 최신 프레임워크로 빠른 페이지 로딩
- **Supabase Realtime**: 투표 결과가 실시간으로 업데이트
- **Tailwind CSS 4**: 깔끔하고 반응형인 디자인
- **Framer Motion**: 부드러운 애니메이션과 파티클 효과
- **모바일 최적화**: 430px 모바일 프레임으로 PC에서도 모바일 경험

## 앞으로의 계획

Thisor는 v1.0 MVP를 출시했지만, 여기서 멈추지 않습니다. 여러분의 피드백을 바탕으로 다음과 같은 기능을 추가할 예정이에요:

- 카테고리 분류 (음식, 패션, 게임 등)
- 인기 투표 랭킹
- 소셜 공유 강화 (카카오톡, 인스타그램)
- 댓글/토론 기능

여러분의 피드백이 Thisor를 더 좋게 만듭니다. 사용하면서 불편한 점이나 추가했으면 하는 기능이 있다면 GitHub 이슈로 알려주세요!
