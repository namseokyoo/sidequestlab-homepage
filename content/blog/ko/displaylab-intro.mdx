디스플레이 분야에서 일하시는 분이라면 한 번쯤 이런 경험이 있으실 겁니다. 색역 분석을 하려면 고가의 소프트웨어 라이선스가 필요하고, 간단한 색 좌표 변환 하나를 위해서도 복잡한 설치 과정을 거쳐야 하는 상황 말입니다.

**Display Lab**은 이런 불편함에서 출발한 프로젝트입니다. 디스플레이 엔지니어와 연구자가 브라우저만 열면 바로 사용할 수 있는 전문가급 분석 도구를 만들고자 했습니다.

## 왜 Display Lab인가?

디스플레이 산업에서 널리 사용되는 도구들이 있습니다. Fluxim의 Phelos, Portrait Displays의 CalMAN 같은 소프트웨어는 강력하지만, 몇 가지 공통적인 진입 장벽이 존재합니다.

- **높은 라이선스 비용**: 연간 수백만 원에 달하는 비용은 개인 연구자나 소규모 팀에게 부담입니다
- **설치와 환경 설정**: 특정 OS에서만 동작하거나, 복잡한 설정 과정이 필요합니다
- **공유의 어려움**: 분석 결과를 동료에게 공유하려면 같은 소프트웨어가 설치되어 있어야 합니다

Display Lab은 이 세 가지 문제를 동시에 해결합니다. **무료이고, 웹 브라우저에서 바로 실행되며, URL 하나로 결과를 공유**할 수 있습니다. 그러면서도 분석의 정밀도는 기존 전문 도구에 뒤지지 않습니다.

## 주요 기능 소개

Display Lab은 디스플레이 평가와 색과학 분야의 핵심 분석 기능 8가지를 제공합니다.

### Color Gamut Analyzer

CIE 1931 및 CIE 1976 색도 다이어그램 위에 패널의 색역을 시각화합니다. sRGB, DCI-P3, BT.2020 등 주요 표준 색역과의 커버리지를 정량적으로 비교할 수 있으며, 여러 패널의 색역을 동시에 오버레이하여 직관적인 비교가 가능합니다.

### Viewing Angle Analyzer

디스플레이의 시야각 특성을 분석합니다. 각도에 따른 색 변화(color shift)와 휘도 변화를 그래프로 확인할 수 있어, OLED와 LCD 패널의 시야각 성능을 객관적으로 비교하는 데 유용합니다.

### HDR Analyzer

HDR 디스플레이의 핵심 성능 지표를 분석합니다. ST.2084(PQ) 커브 기반의 EOTF 특성, 피크 휘도, 명암비, 톤 매핑 정확도 등을 종합적으로 평가할 수 있습니다.

### Spectrum Analyzer

스펙트럼 데이터를 입력하면 파장별 강도 분포를 시각화하고, 색 좌표와 색온도를 자동으로 계산합니다. CSV 파일 업로드를 지원하여 측정 장비에서 바로 데이터를 가져올 수 있습니다.

### CRI/TLCI/TM-30 Calculator

광원의 연색성을 평가하는 세 가지 주요 지표를 한 번에 계산합니다. CRI(Ra)는 물론, 방송 조명에 특화된 TLCI, 그리고 최신 연색성 평가 방법론인 IES TM-30까지 지원합니다.

### Panel Comparator

두 개 이상의 디스플레이 패널을 나란히 놓고 주요 스펙을 비교할 수 있습니다. 색역, 휘도, 명암비, 응답 속도 등 핵심 항목별 비교표를 자동으로 생성합니다.

### Universal Converter

색과학 분야에서 자주 필요한 단위 변환을 지원합니다. CIE XYZ, xyY, L\*a\*b\*, L\*u\*v\*, RGB 간의 변환은 물론, 색온도(CCT)와 듀브(Duv) 계산까지 한 곳에서 처리할 수 있습니다.

### Color Science Calculator

색과학 계산에 필요한 다양한 도구를 모았습니다. 색차(Delta E) 계산, 흑체 궤적(Planckian locus) 위의 색 좌표 계산, 메타메리즘 분석 등 연구와 실무에서 빈번하게 사용하는 계산 기능을 제공합니다.

## 기술적 토대

Display Lab은 SidequestLab의 이전 프로젝트인 **ISCV(Interactive Spectrum & Color Visualizer)**에서 축적한 기술 자산을 적극 활용했습니다. ISCV에서 검증된 스펙트럼 시각화 엔진과 색 좌표 계산 알고리즘을 기반으로, 더 넓은 범위의 디스플레이 분석 기능을 구축했습니다.

기존 코드를 단순히 복사한 것이 아니라, 모듈화와 추상화를 통해 재사용 가능한 컴포넌트로 재설계했습니다. 이는 SidequestLab이 추구하는 "프로젝트 간 기술 자산 재활용" 원칙의 대표적인 사례입니다.

현재 기술 스택은 다음과 같습니다.

- **React 19**: 최신 React의 동시성 기능을 활용한 반응형 UI
- **Vite 7**: 빠른 개발 환경과 최적화된 빌드
- **TypeScript**: 색과학 계산의 타입 안전성 보장
- **D3.js**: 색도 다이어그램, 스펙트럼 차트 등 전문 시각화
- **Tailwind CSS**: 일관된 디자인 시스템

## 향후 계획

현재 Display Lab은 Phase 2-B까지 완료된 상태이며, 다음 단계인 Phase 2-C에서는 더 강력한 기능들을 추가할 예정입니다.

- **데이터베이스 연동**: 분석 결과를 저장하고 이력 관리가 가능하도록
- **PDF 리포트 생성**: 분석 결과를 전문적인 보고서 형태로 내보내기
- **API 제공**: 외부 시스템과의 연동을 위한 RESTful API
