# PRD: SidequestLab 홈페이지

**버전**: v1.1
**작성일**: 2026-01-27
**작성자**: CEO Agent
**상태**: 승인됨

---

## 1. 개요 (Overview)

### 1.1 프로젝트 목적
SidequestLab의 공식 홈페이지를 제작하여:
- 회사 브랜드 정체성 확립
- 프로젝트 포트폴리오 쇼케이스
- 블로그를 통한 정보 공유 및 SEO
- 다국어(한/영) 지원으로 글로벌 접근성 확보

### 1.2 타겟 사용자
- SidequestLab 서비스에 관심 있는 일반 사용자
- 프로젝트/서비스를 찾는 잠재 사용자
- 회사 정보를 확인하려는 파트너/투자자

### 1.3 성공 지표
| 지표 | 목표 |
|------|------|
| MVP 배포 | 1일 내 완료 |
| 페이지 로드 | 3초 이내 |
| Lighthouse 점수 | 90+ |

## 2. 기능 요구사항 (Functional Requirements)

### FR-001: 메인 페이지 (홈)
- **설명**: 회사 소개 및 주요 프로젝트 하이라이트
- **우선순위**: 필수
- **인수 조건**:
  - [ ] 회사 슬로건/미션 표시
  - [ ] 주요 프로젝트 3개 카드 형태로 표시
  - [ ] 각 프로젝트 클릭 시 상세 페이지 또는 외부 링크 이동

### FR-002: 프로젝트 페이지
- **설명**: 모든 프로젝트 목록 및 상세 정보
- **우선순위**: 필수
- **인수 조건**:
  - [ ] 프로젝트 목록 그리드 표시
  - [ ] 각 프로젝트: 이름, 설명, 기술스택, 링크
  - [ ] 프로젝트 데이터: Todo App, Pomodoro Timer, 엔빵 계산기

### FR-003: 블로그 기능
- **설명**: 마크다운 기반 블로그 시스템
- **우선순위**: 필수
- **인수 조건**:
  - [ ] 블로그 목록 페이지 (/blog)
  - [ ] 블로그 상세 페이지 (/blog/[slug])
  - [ ] 마크다운 파일 기반 콘텐츠 관리
  - [ ] 작성일, 제목, 요약 표시
  - [ ] 다국어 블로그 포스트 지원

### FR-004: 다국어 지원 (i18n)
- **설명**: 한국어/영어 언어 전환
- **우선순위**: 필수
- **인수 조건**:
  - [ ] 언어 선택 UI (헤더에 토글/드롭다운)
  - [ ] URL 기반 라우팅 (/ko/*, /en/*)
  - [ ] 모든 정적 텍스트 번역
  - [ ] 기본 언어: 한국어

### FR-005: 회사 소개 페이지
- **설명**: SidequestLab 상세 소개
- **우선순위**: 필수
- **인수 조건**:
  - [ ] 회사 비전/미션
  - [ ] 연혁 또는 주요 마일스톤
  - [ ] 연락처 정보

### FR-006: 반응형 디자인
- **설명**: 모바일/태블릿/데스크톱 지원
- **우선순위**: 필수
- **인수 조건**:
  - [ ] 모바일 네비게이션 (햄버거 메뉴)
  - [ ] 모든 페이지 반응형

## 4. 기술 요구사항

### 4.1 기술 스택
- **Framework**: Next.js 15+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **i18n**: next-intl (확정)
- **Blog**: @next/mdx (확정)
- **배포**: Vercel

### 4.2 프로젝트 구조 (예상)
```
src/
├── app/
│   ├── [locale]/
│   │   ├── page.tsx (홈)
│   │   ├── projects/
│   │   ├── blog/
│   │   └── about/
│   └── layout.tsx
├── components/
├── content/
│   ├── blog/
│   │   ├── ko/
│   │   └── en/
│   └── projects/
├── lib/
└── messages/ (i18n)
    ├── ko.json
    └── en.json
```
